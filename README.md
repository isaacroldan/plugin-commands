# Usage

<!-- usage -->

```sh-session
$ npm install -g @oclif/plugin-commands
$ oclif-example COMMAND
running command...
$ oclif-example (--version)
@oclif/plugin-commands/3.2.1 linux-x64 node-v18.19.1
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
